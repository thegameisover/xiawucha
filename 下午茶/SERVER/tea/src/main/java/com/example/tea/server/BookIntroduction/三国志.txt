三国时代是中国历史上极为分裂混乱的时期，也是人才辈出，英雄崛起，豪强争夺，人气旺盛的时代。

魏、蜀、吴三国的对峙与豪杰俊彦的消长，像一出屡现高潮的连续剧，翻搅出人类最真诚、最私心的情感。

《三国志》与《史记》、《汉书》、《后汉书》并列四史，自有其史学上的价值，相比《三国演义》更加贴近真实。

而致力于用漫画推广古代经典的漫画家蔡志忠，用他的生花妙笔对《三国志》来了一次生动通俗的再现和推广。

"三分"天下，乱世英豪，在蔡志忠的笔下活灵活现，纵横古今。他从小说版的情节出发，融入"蔡子版"的趣味联想和时事妙解，而借着这些人物对社会暗面的著墨，更是刻划细微，入木"三分"。

不管你对《三国志》、《三国演义》有多熟悉，相信我，这绝对是一本你从未见过的"忠肝义胆群英会"
